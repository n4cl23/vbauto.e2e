# Automação Cypress - VBAuto

Projeto de automação E2E utilizando Cypress.

## Tecnologias

- Cypress
- Node.js
- GitHub Actions

## Instalação

npm install

## Executar testes

Abrir interface

npx cypress open

Rodar headless

npx cypress run

## Estrutura

actions → lógica de interação  
pages → seletores  
fixtures → dados de teste  
e2e → cenários de teste
